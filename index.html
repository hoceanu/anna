Ce vei invata ?

HTML este standardul actual pentru structurarea și prezentarea conținutului pe web. Învățarea HTML5 implică înțelegerea modului în care se construiește "coloana vertebrală" a paginilor web moderne, punând accent pe semantică, multimedia și funcționalități interactive.

CSS5 (Cascading Style Sheets) este limbajul folosit pentru a stila și a aranja elementele HTML într-o pagină web. Învățarea CSS implică stăpânirea modului în care culorile, fonturile, layout-ul și poziționarea elementelor sunt gestionate pentru a crea site-uri atractive și responsive.

Înveți JavaScript (JS) pentru a crea pagini web interactive, aplicații web și, mai recent, chiar aplicații backend. Este un limbaj esențial, considerat accesibil pentru începători, care permite manipularea directă a conținutului din browser
